# java-cli-gradle-spring-jacoco-jbehave-junit5-car-data

## Description
A POC for a spring app using junit5,
jacoco, jbehave framework.

## Tech stack
- java
- gradle
	- spring
  - junit5
  - jbehave
  - jacoco

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
